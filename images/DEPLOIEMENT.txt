====================================================
OPTION 1 — NETLIFY (RECOMMANDÉ, GRATUIT)
====================================================
1. Crée un compte sur netlify.com
2. Clique "Add new site" → "Deploy manually"
3. Crée un dossier "verttige972" sur ton ordinateur
4. Mets index.html + le dossier images/ dedans
5. Glisse-dépose ce dossier sur Netlify
6. Le site est en ligne en 30 secondes sur une URL Netlify

CONNECTER TON DOMAINE SUR NETLIFY :
- Site settings → Domain management → Add custom domain
- Tape : elagageabattagemartinique.com
- Netlify te donne 2 DNS à configurer chez ton registrar

====================================================
OPTION 2 — HÉBERGEMENT OVH / IONOS
====================================================
1. Connecte-toi à ton espace client OVH/Ionos
2. Va dans "Hébergement" → "FTP" ou "Gestionnaire de fichiers"
3. Upload index.html à la racine (public_html/ ou www/)
4. Upload le dossier images/ au même endroit
5. Ton domaine pointe automatiquement

====================================================
DNS — CONNEXION DOMAINE VERS NETLIFY
====================================================
Chez ton registrar (GoDaddy, OVH, Namecheap…) :
Ajoute ces enregistrements DNS :

Type A     | @   | 75.2.60.5
Type CNAME | www | [ton-site].netlify.app
